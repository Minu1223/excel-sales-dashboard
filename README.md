# Sales Dashboard using Excel

## Problem Statement
The objective of this project is to analyze sales performance and identify trends to support business decision-making.

## Dataset
Sales transaction data including:
- Date
- Region
- Product
- Revenue
- Quantity

## Tools Used
- Microsoft Excel
- Pivot Tables
- XLOOKUP
- Charts & Dashboards

## Key Insights
- Top 3 products contributed 45% of total revenue
- West region showed highest month-over-month growth
- Discounts negatively impacted profit margins

## Business Recommendations
- Increase inventory for top-performing products
- Optimize discount strategy in low-margin regions
